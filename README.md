## Setup

1. Clone the repo:
   git clone <your-repo-link>

2. Go to the project folder:
   cd <project-folder>

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open in browser:
   Usually http://localhost:5173
